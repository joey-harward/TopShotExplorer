# Top Shot Explorer
This is a sample app to highlight interactions with Dapper's [Top Shot API](https://developers.nbatopshot.com/docs/Introduction).

This app adheres to Android best practices including:
- Compose
- Hilt
- State Flow
- Material3
- Coil
- Media3
- Data Store
- Unit Testing
