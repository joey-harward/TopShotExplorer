# Top Shot Explorer
This is a sample program to highlight some of Dapper's [Top Shot API](https://developers.nbatopshot.com/docs/Introduction).

Using Android best practices and latest tech, here are some highlights:
- Compose
- Hilt
- State Flow
- Material3
- Coil
